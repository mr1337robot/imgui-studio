# Blinded Human Review Rubric

Reviewers receive the reference image, canonical final capture, and animation filmstrip without run
identity. Score each dimension from 1 (unacceptable) to 5 (release quality): visual hierarchy,
consistency, custom appearance, typography and asset quality, animation polish, and overall
perceived quality.

A score of 4 means the work is cohesive, clearly intentional, immediately distinguishable from
stock Dear ImGui, and has no material visual defect. A score of 5 is exceptional reference-quality
work. Reviewers must provide a short rationale. Ratings that differ by more than one point require
an independent adjudication entry before the run can count toward the release gate.

Technical failures are not repaired through human scoring. A run counts as completed only when its
required controls, scenarios, inspection diagnostics, deterministic traces, verified export, and
browser/native parity all pass.
